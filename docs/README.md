## üöÄ Quick Start

### New Developer Onboarding

1. **Setup Environment**
   ```bash
   git clone https://github.com/yourusername/lyne.git
   cd lyne
   npm install
   ```

2. **Read Core Documentation**
   - Start with [Architecture](./ARCHITECTURE.md) for system overview
   - Review [Contributing Guide](./CONTRIBUTING.md) for workflow
   - Check [Testing Guide](./TESTING.md) for testing practices

3. **Run the Project**
   ```bash
   npm run dev      # Start development server
   npm test         # Run tests
   npm run build    # Build for production
   ```

4. **Make Your First Contribution**
   - Pick an issue from GitHub
   - Create a branch
   - Make changes with tests
   - Submit a pull request

## üéØ Common Tasks

### Running Tests

```bash
# Run all tests
npm test

# Run in watch mode
npm run test:watch

# Generate coverage report
npm run test:coverage

# Run specific test file
npm test -- Hero.test.tsx
```

**Learn more**: [Testing Guide](./TESTING.md)

---

### Making Changes

```bash
# Create feature branch
git checkout -b feature/your-feature

# Make changes and test
npm test

# Commit with conventional commits
git commit -m "feat(component): add new feature"

# Push and create PR
git push origin feature/your-feature
```

**Learn more**: [Contributing Guide](./CONTRIBUTING.md)
